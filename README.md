# Library-Management-System
A Python-based Library Management System that manages books, members, book issuing, returns, and fine calculations using OOP and File Handling.

Discription :
The Library Management System is a Python application developed to automate library operations such as managing books, issuing and returning books, searching the catalog, and calculating overdue fines. The project applies Object-Oriented Programming, file handling, and modular programming to create a structured and user-friendly application.

Features :
Add New Books,
View Available Books,
Search Books,
Issue Books,
Return Books,
Fine Calculation,
Record Management,
File Handling / SQLite Support,
